# Task: facilitate-brainstorming-session

## Purpose
Guide the user through a structured brainstorming session to clarify requirements, explore ideas, and define the scope for agent creation or revision.

## Steps
1. State the session topic and objective.
2. Ask open-ended questions to elicit:
   - Agent purpose and goals
   - Key features and capabilities
   - Required tools/integrations
   - Knowledge base and data needs
   - Output requirements
   - User preferences for workflows, commands, and templates
3. Encourage divergent thinking and capture all ideas.
4. Help the user prioritize and refine ideas.
5. Summarize the session outcomes and next steps.
6. Store session notes for inclusion in the agent creation summary.
