## Troubleshooting Support

### ğŸ”§ Common Setup Issues

**File Access Problems**:
- **Issue**: Cannot access architecture file
- **Solutions**: 
  - Verify file path spelling and format
  - Check file permissions and accessibility
  - Ensure file is not locked or in use
  - Try alternative file locations

**Output Folder Issues**:
- **Issue**: Cannot create output folder
- **Solutions**:
  - Check write permissions
  - Verify parent folder exists
  - Try alternative output location
  - Create folders manually if needed

**Path Resolution Problems**:
- **Issue**: Path not found or invalid
- **Solutions**:
  - Use absolute paths instead of relative
  - Check for special characters in path
  - Verify drive letters and network paths
  - Use proper path separators (\ for Windows, / for Unix)
