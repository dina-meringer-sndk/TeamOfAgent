## ğŸš€ Quick Start Guide

### Step 1: Verify Your Environment

First, let me check your current setup. Please run these commands so I can assess what's already installed:

```bash
# Check Node.js version (required: 18+)
node --version

# Check if MCP CLI is available
mcp --version

# Check VS Code version
code --version

# List current VS Code extensions
code --list-extensions | grep -i agent
```

### Step 2: Install MCP Servers

The MCP (Model Context Protocol) servers provide the core team agents functionality:

#### Install MCP CLI (if not already installed):
```bash
npm install -g @modelcontextprotocol/cli
```

#### Set Up MCP Configuration:
I'll help you create the MCP configuration file. The location depends on your OS:

**Windows**: `%APPDATA%\Code\User\globalStorage\rooveterinaryinc.roo-cline\settings\cline_mcp_settings.json`
**macOS**: `~/Library/Application Support/Code/User/globalStorage/rooveterinaryinc.roo-cline/settings/cline_mcp_settings.json`
**Linux**: `~/.config/Code/User/globalStorage/rooveterinaryinc.roo-cline/settings/cline_mcp_settings.json`

### Step 3: Install VS Code Extension

Navigate to the VS Code extension directory and install:

```bash
cd "team-agents-extension"
npm install
npm run compile
```

Then install the extension in VS Code:
```bash
# Package the extension
npm run package

# Install the packaged extension
code --install-extension team-agents-extension-1.0.4.vsix
```

### Step 4: Test Your Setup

Once everything is installed, you can test the agents:

#### Test MCP Servers:
```bash
# Test validation strategist
mcp list-tools validation_strategist

# Test agents factory  
mcp list-tools agents_factory
```

#### Test VS Code Extension:
1. Open VS Code
2. Open GitHub Copilot Chat (Ctrl+Shift+I)
3. Try these commands:
   - `@validation_strategist help`
   - `@agents_factory help`
